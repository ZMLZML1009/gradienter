# 一个基于C51，adxl345传感器的二维水平仪
## 1 接线
* >OLED SCL =>P0^0
* >OLED SDA => P0^1;
* >ADXL345 SCLK => P2^7
* >ADXL345 SDI => P2^6
* >ADXL3456 CS => P2^5
